# DentFlow - Dental Practice Management

A modern dental clinic management platform built with React, TypeScript, and Tailwind CSS.

## Features

- **Patient Management** – Track patient records, history, and details
- **Appointment Scheduling** – Calendar-based booking with conflict detection
- **AI Treatment Recommendations** – Intelligent suggestions powered by AI
- **Recurring Appointments** – Set up repeat visits with flexible rules
- **Responsive Dashboard** – Manage your practice from any device

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Lovable Cloud (database, auth, edge functions)
- **Libraries:** React Big Calendar, React Hook Form, Recharts, date-fns

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Install dependencies
npm install

# Start the dev server
npm run dev
```
